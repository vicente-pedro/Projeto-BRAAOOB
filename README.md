# 📋 Sistema de Gerenciamento de Tarefas

## 📖 Sobre o Projeto

O Sistema de Gerenciamento de Tarefas tem como objetivo auxiliar usuários na organização de suas atividades diárias, permitindo o controle eficiente de tarefas, prazos e prioridades.

O projeto busca oferecer uma forma simples e prática de gerenciar atividades, contribuindo para maior produtividade no dia a dia.

---

## ⚙️ Funcionalidades (Planejadas)

* Cadastro de tarefas
* Edição de tarefas
* Exclusão de tarefas
* Marcação de tarefas como concluídas
* Definição de prazos
* Organização por prioridade
* Listagem de tarefas

---
